Official Hand Talk plugin for Wordpress, a portuguese to libras (Brazilian sign language) translator.

== Description ==

#Make your company website available in Libras (Brazilian sign language) with this powerful tool!

###Information to everyone!

There are nearly 10 million deaf people in Brazil (IBGE 2010). Most of them have difficulties understanding Portuguese language and rely on sign language interpretation to understand what is written. 

###Simple and efficient

With the Hand Talk button on your website, the content can be automatically translated from Portuguese to Brazilian Sign Language and available to the deaf community in Brazil.

== Installation ==

1. Upload `wpht` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to http://handtalk.me/sites to subscribe, copy the token string found on the script and fill the space on the plugin settings to activate it.

Technical notice: This plugins includes a third part javascript file from our server in order to work. Your token is send to us in order to validate if your plan is currently active. No further information is send or collected by our services.
